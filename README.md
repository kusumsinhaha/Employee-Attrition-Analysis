# Employee Attrition Analysis & Prediction

## Project Description
Analyzing HR data to predict employee attrition and provide actionable insights for HR decisions.  
This project includes data cleaning, EDA, feature engineering, predictive modeling, and visualization.

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-Learn, Matplotlib, Seaborn)
- Jupyter Notebook
- Power BI / Tableau (optional)

## Project Sections
1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Predictive Modeling
5. Feature Importance & Visualization

## Dataset
A sample HR dataset with employee info, job satisfaction, income, tenure, and attrition.

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run each Python script in order from `1_data_cleaning.py` to `5_visualization.py`
